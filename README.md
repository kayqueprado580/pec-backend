# Personal Economy Control - Backend

Projeto dedicado a persistencia de dados para o sistema de finanças PEC

## Funcionalidades

* Autenticação de Usuários
* Persistencia de dados Usuários
* Persistencia de dados Registros
* Persistencia de dados Categorias

## Tipo Aplicação
* API HTTP
* API BFF

## Stack
* Javascript
* Typescript
* NodeJS
* NestJS

## Instalação

```bash
$ npm install
```

## Rodando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
